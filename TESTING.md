# 🧪 Testing Guide

## 🛠️ Initial Setup

Copy environment variables for testing

```bash
cp .env.test.example .env.test
```

## 🔬 Unit Tests

### Backend Tests
```bash
npm run test:unit:backend
```

### Frontend Tests
```bash
npm run test:unit:frontend
```

## 🔗 Integration Tests

```bash
npm run test:integration:backend
```
